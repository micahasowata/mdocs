## Endpoint
POST https://tapi.micahasowata.com/tasks/{task_id}/update

## Headers
Content-Type: application/json

## Requires Authentication
Yes

## Path Parameters
task_id | id of task to be updated

## Request Body
```json
{
  "title": "Run today",
  "description": "Do 10km by 5:05pm"
}
```
