## Endpoint
POST https://tapi.micahasowata.com/tasks/create

## Headers
Content-Type: application/json

## Requires Authentication
Yes

## Request Body
```json
{
  "title": "Run today",
  "description": "Do 10km by 5:05pm"
}
```

## Response

<Tabs>
  <TabItem label="201 Created">
    ```json
    {
      "payload": "id_of_task"
    }
    ```
    </TabItem>
  <TabItem label="422 Unprocessable Entity">
    ```json
    {
      "error": {
        "title": "must not be empty"
      }
    }
    ```
    :::tip
      Ensure both title and description fields are required in the input form and you would be fine
    :::
  </TabItem>
  <TabItem label="409 Conflict">
    ```json
    {
      "error": "task exist"
    }
    ```
    </TabItem>
</Tabs>
